"Go to Fencing\nr/Fencing\n•\n1y ago\nSuperFencingSystem\nMy macOS scoring machine software \"Schermatica\" (VSM USB adapter/interface compatible) is now available for download!\n\nHello again r/Fencing,\n\nI’m thrilled to announce the release of my macOS & iOS scoring machine apps, ‘Schermatica’ and ‘Schermatica Remote’! Schermatica is macOS scoring machine software, similar to VSM on Windows. And like VSM, it is compatible with the VSM reel & piste USB hardware interface module! Simply plug in the VSM USB box, boot up Schermatica, and enjoy running a mostly-fully-featured scoring machine on a modern Mac.\n\nPlease note that Schermatica is NOT a refined, ready-for-tournament piece of software. It is brand-new, so expect bugs, quirks, and oddities in how it functions. Among the missing feature-set in V1 is automatic passivity card issuing (although you can still turn P-cards on and off), self-start, and compatibility with USB IR remote receivers a la VSM. To control a Schermatica 'machine', you’ll therefore need to use the \"Schermatica Remote\" iOS companion app. Schermatica Remote communicates using encrypted Bonjour messages, so both the machine Mac and the remote controller iPhone must be on the same local network.\n\nWhile not entirely feature-complete, this release version of Schermatica does have functionality similar to VSM, like hit duration and hit difference display, a timing adjustment screen, whipover indication, the ability to customize various sound effects, and piste diagnostics. While my goal isn’t to simply clone VSM 1:1, VSM does have multitudes of features like these that make obvious sense to include in scoring machine software. A big thanks to J. S. Hart for doing it first!\n\nLike my iOS/iPadOS video replay app \"SFS\", Schermatica and Schermatica Remote are completely free to download. MacOS 15 and iOS 18 are required.
